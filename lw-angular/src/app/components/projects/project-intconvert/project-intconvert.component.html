IntConvert

IntConvert was developed with the goal of creating a simple, intuitive, and well-designed application for converting integers with different bases. As a result, with this app, users are able to easily make accurate conversions between the binary, octal, decimal, and hexadecimal number systems.

Goals & Requirements

1

Conversions Between 4 Bases

The main goal of this project was to allow users to convert integers between the binary, octal, decimal, and hexadecimal number systems. This is the minimum viable functionality of the application and was vital to the project. Additionally, I did not want to use libraries for the conversions in order to get practice with programming logic and arithmetic.

2

Formula Calculation

In addition to converting the integers between the number systems, I wanted to be able to show users how to perform the calculations themselves by displaying the math and functions required. Teaching users how to do the conversions on their own was just as important to me as the conversions themselves.

3

Input Validation

I knew there could be issues with users trying to convert numbers that aren't compatible with the different number bases. Implementing validation for my input field was an important requirement for this project.
