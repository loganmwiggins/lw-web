The goal of this project was to design and develop a personal website for myself to promote my professional activity to potential employers. As you can see from this website, I was able to successfully teach myself web development practices while simultaneously creating this product as a personal digital portfolio.

Goals & Requirements

1

Showcase Experience

The main goal for my personal website is to successfully showcase my experience by displaying my resume information, skills, coursework, and projects. The ability to communicate my professional endeavours is crucial to successfully advertising myself to employers through this website.

2

Intriguing User Interface

When starting the web development for this site, I knew I wanted to design and develop a user interface that is immediately eye-catching and would intrigue users to explore the website. I aimed to create a unique UI that would set my site apart from other resume websites, and subsequently, help to set me apart as a candidate.

3

Responsive Design (Mobile UI)

With the unique interface I developed for this website on desktops and laptops, a challenge I faced early on was making the site responsive. One of my original goals was to create a mobile/tablet UI that would be just as pleasing and usable as the desktop version.
