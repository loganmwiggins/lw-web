Goals & Requirements

1

Showcase Experience

The main goal for my personal website is to successfully showcase my experience by displaying my resume information, skills, coursework, and projects. Being able to communicate my professional endeavours is crucial to successfully advertising myself to employers through this website.

2

Intriguing User Interface

When starting the web development for this site, I knew I wanted to design and develop a user interface that is immediately eye-catching and would intrigue users to explore the website. I aimed to create a unique UI that would set my site apart from others', and subsequently, help set me apart as a candidate.

3

Responsive Design (Mobile UI)

With the unique interface I designed and developed for the website on desktops and laptops, I knew it would be challenging to make the site responsive. One of my original goals was to create a mobile/tablet UI that would be just as pleasing and usable as the desktop version.
